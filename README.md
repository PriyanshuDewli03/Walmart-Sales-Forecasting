# 🛒 Walmart Weekly Sales Forecasting using Prophet

## 📌 Project Overview

This project focuses on forecasting Walmart's weekly sales using historical retail data. The goal is to build a time-series forecasting model that helps predict future sales trends, enabling better inventory management, demand planning, and business decision-making.

The project follows an end-to-end data science workflow, including data preprocessing, exploratory data analysis (EDA), feature engineering, visualization, forecasting with Facebook Prophet, and business insights.

---

## 🎯 Business Problem

Retail businesses need accurate sales forecasts to optimize inventory levels, reduce stock shortages, and minimize excess inventory.

Using historical Walmart sales data, this project develops a forecasting model that predicts future weekly sales while identifying trends, seasonality, and the impact of holidays.

## 🔮 Forecasting Model

The forecasting model was developed using **Facebook Prophet**, a robust time-series forecasting library designed for business forecasting problems.

The model captures:

- Overall trend
- Weekly patterns
- Yearly seasonality
- Holiday effects
- Future sales predictions
